## 介绍

自动识别第三方网站的媒体（图片、视频、音频）并内嵌到 Flarum 。

## 安装 暂不可用

```
composer require zzzz0317/flarum-ext-mediaembed
```

## 自动识别图片链接

- 直接插入图片链接即可。

## 支持自动识别媒体的网站

- YouTube

- 网易云音乐

- 哔哩哔哩
